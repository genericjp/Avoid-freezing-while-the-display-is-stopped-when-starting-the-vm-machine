Avoid freezing while the display is stopped when starting the vm machine

PHENOMENON
CONFIGURATION COMPARISON

PHENOMENON

  • In s74-1, when the vm machine starts, the vm machine freezes with the display stopped.
  • In s74-2, such a phenomenon is not seen and the vm machine starts normally and functions normally.

  • Both s74-1 and s74-2 are windows 11 (home), and the configuration of vbox is almost the same.

WORKAROUND

  1. Suppress Core Isolation.

1) confirm Device security that a warning(⚠) is not displayed.

Settings/Privacy & security/Windows Security/Device security

2) Start Registry Editor (regedit).
3) Change Device Gurad's Enabled to turn off (1-> 0). 


Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\DeviceGuard\Scenarios\HypervisorEnforcedCodeIntegrity
Enabled = 0

4) Restart system. 

             restart

AVOIDANCE CONFIRMATION

  1. Check Core Isolation.

1) After restarting, confirm that confirm Device security under Windows Security that a warning(⚠) is displayed. 

Settings/Privacy & security/Windows Security

2) Confirm that confirm Core isolation under Device security that a warning(⚠) is displayed. 

Settings/Privacy & security/Windows Security/Device security

3) Confirm that confirm Memory Integrity under Core isolation that a warning(⚠) is displayed and the state is off. 

Settings/Privacy & security/Windows Security/Device security/Core isolation

EVIDENCE CONFIRMATION WHEN PHENOMENON OCCURS

1) Right-click on the vm machine and use vm machine Show log ... to confirm that ERROR is displayed in the vm machine log.

ESPECIALLY NOTEWORTHY

Create a vm machine with Core Isolation suppressed and it will not freeze at startup.
If you start the vm machine with Core Isolation restored to its original on state, there seems to be a new problem that booting is possible but not progressing.
In that case, it is necessary to recreate the vm machine with Core Isolation suppressed.

In addition, it seems that the same problem will occur with the vm machine acquired by migrating with Virtual Appliance from a device that has suppressed or does not need to suppress Core Isolation.
